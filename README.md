# Machine-Learning-Assignment-2

## Developing an interactive application to enable the user to upload dataset and then select the model for classification and deploying using Stream-lit.

### The models to be deployed
1. Logistic Regression
2. Decision Tree Classifier
3. K-Nearest Neighbor Classifier
4. Naive Bayes Classifier - Gaussian or Multinomial
5. Ensemble Model - Random Forest
6. Ensemble Model - XGBoost

### The evaluation metrics
1. Accuracy
2. AUC Score
3. Precision
4. Recall
5. F1 Score
